<p><i>Squeezelite</i> - це аудіоплеєр без графічного інтерфейсу для Lyrion Music Server (LMS) - <a href='https://github.com/LMS-Community/slimserver'>Squeezelite</a>, портований на Android. Для його використання вам буде потрібен доступ до сервера LMS у вашій мережі; див. розділ <a href='https://lyrion.org/getting-started/'>Початок роботи з LMS</a>.</p>
