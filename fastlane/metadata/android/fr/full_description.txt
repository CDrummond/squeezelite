<p><i>Squeezelite</i> est un lecteur audio "sans interface" pour <a href='https://lyrion.org/'>Lyrion Music Server (LMS)</a>. Un serveur LMS doit être présent sur votre réseau pour utiliser cette application (plus d'info <a href='https://lyrion.org/getting-started/'>ici</a>).</p><p>La lecture de la musique peut être pilotée depuis un navigateur accédant au serveur LMS ou depuis une application dédiée (voir cette <a href='https://lyrion.org/players-and-controllers/#software-based-controllers'>liste de "contrôleurs"</a> ou directement cette <a href='https://github.com/CDrummond/lms-material-app'>application du même développeur</a> si <a href='https://github.com/CDrummond/lms-material'>"Material Skin"</a> est installé sur votre serveur LMS).</p><p>Cette application est une adaptation pour Android de l'application <a href='https://lyrion.org/players-and-controllers/squeezelite/'>Squeezelite historique</a>.</p>
